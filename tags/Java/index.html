Java并发编程之AQS

什么是 AQS

同步工具类 也叫同步器（Synchronizer）。在使用同步器时，我们发现不同的同步器存在许多共同点，例如 ReentrantLock 和 Semaphore 都支持每次允许一定数量线程通过/等待/取消，也都支持让等待线程执行公平或非公平的队列操作等。

事实上，很多同步工具类在实现时都使用了共同的基类，这就是 AbstractQueuedSynchronizer（AQS），抽象队列同步器。
